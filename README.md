# GDG Backend

MERN Stack Backend with Express.js and MongoDB

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Create `.env` file with MongoDB URI and PORT

3. Start the server:
   ```bash
   npm start
   ```

## Project Structure

- `models/` - Mongoose schemas
- `routes/` - API routes
- `controllers/` - Route handlers
- `server.js` - Main application file
